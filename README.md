# 文档
- [部署文档]  开箱即用的小程序前后台，后台使用idea开发，前端使用微信开发者工具开发。
- [后台部署]  详见党建后台系统 readme 文件

### 项目架构
- SpringBoot2.0 mysql数据库
-  [html2wxml ](https://gitee.com/qwqoffice/html2wxml) 用于微信小程序的HTML和Markdown格式的富文本渲染组件，支持代码高亮部署小程序需要集成插件
